
VS中webservice添加web引用

将 Web 引用添加到项目
1.在“解决方案资源管理器”中，右击要添加服务的项目的名称，然后单击“添加服务引用”。
将出现“添加服务引用”对话框。
2.
在“添加服务引用”对话框中，单击“高级”按钮。
将出现“服务引用设置”对话框。
3.
在“服务引用设置”对话框中，单击“添加 Web 引用”。
将出现“添加 Web 引用”对话框。
4.
在“URL”框中输入要使用的 Web 服务的 URL。如果不知道 URL，则请使用浏览窗格中的链接查找所需的 Web 服务。
